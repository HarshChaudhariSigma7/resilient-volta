Cascade

Cascade is a physics-informed neural network built to reduce error in turbulence closure for coupled thermal-fluid-structural simulations (the class of problems where the governing equations are known but standard models still diverge from reality under combined loading).

It's trained to correct, not replace, existing solvers, and it's being built with the same discipline as everything else on this site: every output will state its confidence, and where the model is extrapolating past validated conditions, it will say so rather than smooth it over.
